# Сайт-визитка

Одностраничное портфолио / визитка. Статический HTML + CSS.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 8000
# или: npx serve .
```

## Секреты

В репозитории нет и не должно быть секретов. Переменные окружения и ключи храните локально (файлы `.env*` в `.gitignore`).
